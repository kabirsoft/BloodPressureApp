---------------------------- Documentation ----------
--------1. Description:(what this app do)
Take BloodPressue(systalic and diastolic) input value, analysis data and then app will show the stage of your body. Is it normal or critical or you may need to change your life style

------- 2. How to run:
i) Download from github
ii)You have to migrate database to run this application.
VS->Tool->nuget package manager->package manager console -> select default project: BloodPressureApp.Data
>add-migrate "Create database BloodPressureDB"
>update-database

-------------- 3. Part --------
This project has four(4) parts.
BloodPressureApp.Data: is data layer. Both MVC and Webservice get data from this layer.
BloodPressureApp(MVC) : This is MVC project. Controller from here read/save the data to/from BloodPressureApp.Data and send data to View
BloodPressureApp.WebService (API) : This is RestFulAPI. Controller from webservice Get/post data to/from BloodPressureApp.Data
BloodPressureApp.UnitTest: Unit test for BloodPressureApp.WebService's controller function (Get, Post, Delete)

---------------- 4.How to run MVC Project ------
set as startup project : BloodPressureApp 
ctr + f5 or VS->Debug->start without debugging

------------ 5.How to run WebAPI project --------
set as startup project : BloodPressureApp.WebService 
ctr + f5 you will see Get functions value.
URL: https://localhost:44394/api/Measurements  //it will return data in json formatt
PostMan:
GET: https://localhost:44394/api/Measurements 
post:  {
        "systolic": 135,
        "diastolic": 85      
       }
DELETE: https://localhost:44394/api/Measurements/5 //5 is id here 

-----------------6. Functionality ------------
i. Login functionality
ii) Save, Read, Delete data (both by MVC and WebAPI), data save in Database
iii) Sort by click on header column
